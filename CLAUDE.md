# CLAUDE.md

## Design Context

This is a static HTML/CSS/JS marketing landing page (no build framework) for
Sooncar, a taxi app serving Astana and Kazakhstan. Register: **brand**.
Platform: **web**.

- `PRODUCT.md` — who it's for (passengers + driver-partners, equal weight),
  purpose, positioning, brand personality, anti-references, accessibility.
- `DESIGN.md` — visual system: OKLCH-style tokens, Montserrat/Inter type
  pairing, restrained neutral palette + single accent, 2-moment motion budget.

Key rule for any change here: this is a single-minded download page — every
section should move toward the app-store download CTA, with passenger and
driver-partner messaging always sized and ordered equally.

Read `PRODUCT.md` and `DESIGN.md` before making design changes.
